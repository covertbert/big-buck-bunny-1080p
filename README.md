big-buck-bunny-1080p-30s
========================

30 first seconds of Big Buck Bunny (1080p)

This is intended to be used as a sample video for testing (ex: `<video>` tag).

Installation
------------

Install using NPM:

    npm install big-buck-bunny-1080p-30s
